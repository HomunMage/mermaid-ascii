text-graph: DSL text input → ASCII/Unicode graph output

Usage: text-graph [OPTIONS] [INPUT]

Arguments:
  [INPUT]  Input file (reads from stdin if omitted)

Options:
  -a, --ascii            Use plain ASCII characters instead of Unicode box-drawing
  -d, --direction <DIR>  Override graph direction (LR, RL, TD, BT)
  -p, --padding <N>      Node padding (spaces inside node border on each side) [default: 1]
  -o, --output <FILE>    Write output to this file instead of stdout
  -h, --help             Print help
  -V, --version          Print version
